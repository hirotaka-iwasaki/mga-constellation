## 運用の基本方針

### アカウントのコンセプト

**「ミセスの楽曲を星座で可視化するファンメイドプロジェクト」**

- 楽曲データやライブセトリを視覚的に楽しめるユニークな切り口
- ファン同士が「この星座、この曲順いいよね」と共感できる場
- 非営利・ファン活動であることを明確に

### ターゲット

- Mrs. GREEN APPLEのファン（ライト層〜コア層）
- ライブ参戦者（セトリの振り返りに関心）
- 楽曲の繋がりやコンセプトに興味がある層

### トーン＆マナー

- 丁寧で親しみやすい口調
- 過度なテンションは避ける
- アーティストへのリスペクトを常に意識
- ファン同士の交流を促す

## プラットフォーム別戦略

### X（旧Twitter）

**役割**: リアルタイム情報発信・ファン交流・拡散

**特徴**:
- 拡散力が最も高い
- リアルタイム性（ライブ当日の反応など）
- テキスト中心だが画像・動画の効果大

**投稿タイプ**:

| タイプ | 頻度 | 例 |
|--------|------|-----|
| サイト更新告知 | 随時 | 「新機能追加しました」 |
| 楽曲トリビア | 週2-3回 | 「この曲とあの曲、実は同じ日リリース」 |
| ライブ関連 | ライブ前後 | セトリ振り返り、星座表示 |
| 交流投稿 | 週1-2回 | 「好きなアルバム曲順、教えて」 |

**運用Tips**:
- **投稿30分以内のエンゲージメント**が重要（リプ・RT・いいね）
- 画像付き投稿を基本にする（星座のスクリーンショットなど）
- ハッシュタグは2-3個に絞る（#ミセス #MrsGREENAPPLE など）
- ファンアカウント（PCF）ラベルの設定を検討

**最適な投稿時間**:
- 朝 7:00-8:00（通勤時間）
- 昼 12:00-13:00（昼休み）
- 夜 20:00-22:00（ゴールデンタイム）

### Instagram

**役割**: ビジュアル訴求・ブランディング

**特徴**:
- 2025年は「関係性の深さ」重視のアルゴリズム
- DMシェアされる投稿が強く評価される
- フィード投稿が再評価されている

**投稿タイプ**:

| タイプ | 形式 | 頻度 |
|--------|------|------|
| 星座ビジュアル | フィード | 週2-3回 |
| サイト使い方 | リール | 月2-3回 |
| ストーリーズ | 日常・質問 | 毎日〜隔日 |

**コンテンツ案**:

1. **星座ビジュアル投稿**
   - アルバムごとの星座を美しく切り取った画像
   - ライブセトリの星座ライン
   - 比較（昨年vs今年のツアーセトリ）

2. **リール動画**
   - サイトの操作デモ（15-30秒）
   - 「1分で分かる○○ツアーのセトリ」
   - 楽曲リリース順のタイムラプス

3. **ストーリーズ**
   - アンケート機能（「次に追加してほしいライブは？」）
   - クイズ（「この星座、何のアルバム？」）
   - カウントダウン（新機能リリースなど）

**運用Tips**:
- **シェアしたくなる投稿**を意識（友達に送りたい、保存したい）
- キャプションにもキーワードを入れる（SEO対策）
- 位置情報やタグを活用
- リールは3分以内、最初の3秒で惹きつける

---

### Threads

**役割**: テキスト中心の交流・新規開拓

**特徴**:
- 2025年現在、最も伸びやすいアルゴリズム
- Instagramとの連携で相乗効果
- 投稿後30分のエンゲージメントが重要

**投稿タイプ**:

| タイプ | 頻度 | 例 |
|--------|------|-----|
| 考察・トリビア | 週3-4回 | 「StaRtとケセラセラ、星座で見ると...」 |
| 質問投げかけ | 週2回 | 「あなたの推し曲の座標、見てみた？」 |
| サイト更新 | 随時 | Xと同内容でOK |

**運用Tips**:
- プロフィールを作り込む（何者か明確に）
- 名前はひらがな・カタカナ推奨（交流しやすい）
- 積極的にフォロー・コメントする
- 今が参入チャンス（先行者利益）

## コンテンツアイデア

### 定番コンテンツ（毎週）

1. **楽曲リリース記念日**
   - 「今日は○○のリリースから○年」
   - 星座マップでの位置を添えて

2. **今週の星座ピックアップ**
   - 特定のアルバムやライブを紹介
   - 収録曲や背景の小ネタ

3. **質問・アンケート**
   - 「このセトリ、あなたは何点？」
   - 「次に見たい機能は？」

### イベント連動コンテンツ

1. **ライブ前**
   - 過去の同会場セトリを紹介
   - 「予想セトリ」を募集

2. **ライブ当日〜翌日**
   - セトリの星座を速報
   - ファンの感想をシェア

3. **新曲・アルバムリリース**
   - 星座マップへの反映を告知
   - 収録順の星座ラインを紹介

### バズを狙えるコンテンツ

1. **意外な発見系**
   - 「同じ日にリリースされた2曲」
   - 「全アルバム1曲目を繋ぐと...」

2. **比較・ランキング系**
   - 「最も曲数の多いライブは？」
   - 「セトリ被り率ランキング」

3. **参加型コンテンツ**
   - 「あなたのベストセトリを作ろう」
   - 「この星座、何のアルバムか当ててみて」

## クロスプラットフォーム戦略

### プラットフォームの役割分担

```
[Threads] 新規開拓・テキスト交流
    ↓ プロフィールリンク
[Instagram] ブランディング・ビジュアル
    ↓ ストーリーズでシェア
[X] 拡散・リアルタイム・ファン交流
    ↓ プロフィールリンク
[星座マップサイト] 本体
```

### 連携のポイント

1. **同じ内容を各プラットフォームに最適化**
   - X: 短いテキスト + 画像
   - Instagram: ビジュアル重視 + 詳細キャプション
   - Threads: 語り口調で深掘り

2. **相互誘導**
   - 「詳しくはInstagramで」「サイトで実際に見てみて」
   - 各プロフィールに他SNS・サイトリンク設置

3. **プラットフォーム固有の強みを活かす**
   - Xの速報性（ライブ直後）
   - Instagramのビジュアル（星座の美しさ）
   - Threadsの会話性（深い考察）

## 成長フェーズ別の施策

### Phase 1: 立ち上げ期（0-500フォロワー）

**目標**: アカウントの土台固め

**やること**:
- プロフィールを作り込む（何者か・何を発信するか）
- 最低30投稿をストックしてから本格始動
- ミセス関連アカウントを積極的にフォロー
- 他ファンの投稿にコメント・リアクション
- バズは狙わず、継続を重視

**KPI**:
- 投稿数（最初の1ヶ月で30投稿）
- フォロー数に対するフォロワー比率

### Phase 2: 成長期（500-3000フォロワー）

**目標**: 認知拡大・エンゲージメント向上

**やること**:
- 投稿頻度を上げる
- 参加型コンテンツを増やす
- 他ファンアカウントとのコラボ
- ライブイベントに合わせた投稿強化
- 分析ツールで効果測定開始

**KPI**:
- エンゲージメント率（いいね+コメント/フォロワー数）
- サイトへの流入数

### Phase 3: 安定期（3000フォロワー〜）

**目標**: コミュニティ化・継続運用

**やること**:
- UGC（ユーザー投稿）のシェア
- 定期企画の確立
- ファン同士の交流促進
- サイトの新機能と連動した施策

**KPI**:
- UGC数
- リピート訪問率

## 投稿の自動化

運用を継続するには、効率化が重要。ツールを活用して負担を減らしつつ、質を維持する。

### 自動化ツール比較

| ツール | 無料プラン | 特徴 | おすすめ用途 |
|--------|-----------|------|-------------|
| **Buffer** | ○（3チャンネル/30投稿） | シンプルUI、予約投稿 | 初心者・個人運用 |
| **SocialDog** | ○（X月5件/合計10件） | X特化、日本語対応 | X中心の運用 |
| **Hootsuite** | ×（2023年終了） | 高機能ダッシュボード | 大規模チーム向け |
| **Later** | ○（限定的） | Instagram特化 | Instagram中心の運用 |

### Buffer（おすすめ）

無料で始めやすく、個人運用に最適。

**無料プランでできること**:
- 3つのSNSアカウント連携
- 合計30投稿までの予約
- ドラッグ&ドロップ対応カレンダー
- AIアシスタント機能（キャプション提案）

**有料プラン**: 月額$6〜（1チャンネルあたり）

**始め方**:
1. [buffer.com](https://buffer.com) でアカウント作成
2. X/Instagram/Threadsアカウントを連携
3. 投稿したい時間帯を設定
4. コンテンツをキューに追加

### SocialDog（X特化）

X（旧Twitter）運用に特化した日本製ツール。

**無料プランでできること**:
- Xアカウント1つ連携
- 月5件の予約投稿（合計10件まで管理）
- 2週間分の分析データ
- フォロワー推移の確認

**活用シーン**:
- 最適な投稿時間の分析
- フォロワー増減の把握
- 予約投稿による定期配信

### ノーコード自動化（Zapier / Make）

より高度な自動化を実現するノーコードツール。

**Zapier**:
- 7000以上のアプリ連携
- 無料: 月100タスク / 5つのZap
- 例: 「サイト更新 → X自動投稿」

**Make（旧Integromat）**:
- 無料: 月1000回実行
- より複雑なワークフローが可能
- 例: 「新曲リリース日 → 複数SNSに同時投稿」

**自動化の例**:
```
[トリガー] サイトの新機能リリース
    ↓
[アクション1] X に告知投稿
[アクション2] Instagram ストーリーズ投稿
[アクション3] Threads に詳細投稿
```

### AI活用（ChatGPT連携）

2025年はAI活用が加速。投稿作成の効率化に活用可能。

**活用できる場面**:
- キャプション・投稿文の下書き作成
- ハッシュタグの提案
- 投稿アイデアのブレスト
- 複数パターンのA/Bテスト案作成

**ChatGPTプロンプト例**:
```
あなたはMrs. GREEN APPLEのファンアカウントのSNS担当です。
以下の条件でX投稿文を3パターン作成してください。

【条件】
- 内容: アルバム「ANTENNA」の星座マップを紹介
- トーン: 親しみやすく、押し付けがましくない
- 文字数: 100-140字
- ハッシュタグ: 2-3個
```

**注意点**:
- AI生成文は必ず人間がチェック・修正
- 機械的な文章にならないよう調整
- ファン活動らしい温かみを保つ

### 自動化の注意点

1. **完全自動化は避ける**
   - リアルタイムの反応・交流は人力で
   - 予約投稿 + 手動対応のハイブリッドがベスト

2. **投稿品質を維持**
   - 自動化で量を増やしても質を落とさない
   - 定期的にエンゲージメントをチェック

3. **トラブル対応**
   - 予約投稿が不適切なタイミングになる可能性
   - 災害・訃報時は予約を一時停止

## SNS広告の活用

非営利のファン活動でも、少額広告で認知拡大を図ることは可能。ただし、公式ガイドラインの「収益化禁止」に抵触しないよう注意が必要。

### 広告を使う目的

- **認知拡大**: サイトやアカウントの存在を知ってもらう
- **コミュニティ拡大**: 同じファン同士を繋げる
- **イベント告知**: ライブ前後の特別コンテンツの周知

**注意**: 収益目的の広告運用は公式ガイドライン違反。あくまで「ファン活動の周知」として。

### X広告

**前提条件**（2025年現在）:
- **Xプレミアム加入が必須**（月額980円〜）
- クレジットカード登録

**最低予算**: 設定なし（数百円から可能）

**メリット**:
- 競合が少なく、低コストでインプレッション獲得
- リツイートによる二次拡散（追加費用なし）
- 既存の投稿をそのまま広告に使える

**おすすめの広告タイプ**:
- **プロモツイート**: 通常投稿を広告として表示
- **フォロワー獲得広告**: アカウント認知向上

**ターゲティング例**:
- 興味関心: 「J-POP」「邦ロック」「ライブ」
- フォロワーが似ているアカウント: ミセス公式、音楽メディア
- 地域: 日本

**始め方**:
1. Xプレミアムに加入
2. [ads.twitter.com](https://ads.twitter.com) にアクセス
3. 広告アカウント作成（クレカ登録）
4. キャンペーン作成 → 目的選択 → ターゲット設定 → 予算設定

### Instagram広告

**前提条件**:
- ビジネスアカウントへの切り替え（無料）
- Facebookページとの連携（推奨）

**最低予算**: 100円/日〜（ただし効果検証には月1万円程度推奨）

**メリット**:
- 高精度なターゲティング（Facebook連携）
- ビジュアルコンテンツとの相性◎
- 「投稿を宣伝」ボタンで簡単に開始可能

**おすすめの広告タイプ**:
- **投稿の宣伝**: 反応の良かった投稿を広告化
- **ストーリーズ広告**: フルスクリーンでインパクト大

**ターゲティング例**:
- 年齢: 15-35歳
- 興味関心: 音楽、ライブ、J-POP
- 類似オーディエンス: 既存フォロワーに似たユーザー

### 少額広告運用のコツ

**予算の目安**:

| フェーズ | 月予算 | 目的 |
|----------|--------|------|
| テスト期 | 3,000-5,000円 | 効果測定、ターゲット検証 |
| 本格運用 | 10,000-30,000円 | 認知拡大、フォロワー獲得 |

**効果を最大化するポイント**:

1. **既存の人気投稿を広告化**
   - オーガニックで反応が良かった投稿を使う
   - 「広告っぽさ」が薄れ、エンゲージメントが高い

2. **A/Bテスト**
   - 複数のクリエイティブを少額でテスト
   - 効果の高いものに予算を集中

3. **期間限定で集中投下**
   - ライブ前後など、関心が高まるタイミング
   - 新機能リリース時

4. **リターゲティング活用**
   - サイト訪問者に再アプローチ
   - Instagram: ピクセル設置が必要

### 広告運用の注意点

1. **公式ガイドラインの遵守**
   - 収益化目的と見なされないよう注意
   - 「非営利のファンプロジェクト」を明記

2. **過度な宣伝感を避ける**
   - ファンコミュニティとしての温かみを保つ
   - 押し付けがましいコピーはNG

3. **効果測定を必ず行う**
   - 投じた費用に対するリターンを確認
   - 効果がなければ撤退も選択肢

4. **税務上の考慮**
   - 個人の趣味として処理可能な範囲で
   - 規模が大きくなる場合は専門家に相談

## 注意事項とガイドライン

### ファン活動ガイドラインの遵守

[公式ガイドライン](https://www.universal-music.co.jp/mrsgreenapple/guide/)に基づき:

- **非営利**: 広告なし・収益化なし
- **公式素材の適切な使用**: 出典明記
- **リスペクト**: アーティストのイメージを損なわない

### 投稿時の注意点

1. **著作権表記を忘れずに**
   - 楽曲著作権はMrs. GREEN APPLEおよび関係者に帰属
   - 「ファンメイドプロジェクト」であることを明記

2. **ネガティブな話題は避ける**
   - 批判・誹謗中傷は絶対NG
   - 炎上している話題に乗らない

3. **情報の正確性**
   - リリース日やセトリは公式情報を確認
   - 誤情報を流した場合は速やかに訂正

4. **他ファンへの配慮**
   - ネタバレ注意（ライブセトリは期間を空けるなど）
   - 意見の押しつけをしない

### 炎上リスク対策

- 政治・宗教・センシティブな話題に触れない
- 他アーティストとの比較をしない
- ファン同士の論争に加担しない
- 問題が起きたら謝罪→削除→沈静化を待つ
